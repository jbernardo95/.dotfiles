# Rectangle

## Commands

- Left Half: `Ctrl + Alt + arrow left`
- Right Half: `Ctrl + Alt + arrow right`
- Top Half: `Ctrl + Alt + arrow up`
- Bottom Half: `Ctrl + Alt + arrow down`
- Maximise : `Ctrl + Alt + f`
- Center: `Ctrl + Alt + c`

All other options unset.

## Settings

- Launch on login: Yes
- Hide menu bar icon: Yes
